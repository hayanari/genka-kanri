# 入札スケジュール機能 説明書

## 概要

入札スケジュール機能は、入札案件の日程と状況を一括管理し、落札や当社受注見込みの案件を案件管理（案件一覧）へ円滑に登録するための機能です。

---

## 主な機能

### 1. 入札スケジュールの登録

「新規追加」ボタンから、以下の項目で入札案件を登録できます。

| 項目 | 説明 | 必須 |
|------|------|------|
| 案件名 | 入札案件の名称（例：○○道路 路面清掃業務） | ✓ |
| 発注者 | 発注先（例：堺市） | ✓ |
| 区分 | 工事 / 業務 | ✓ |
| 入札日 | 入札の日程 | ✓ |
| ステータス | 入札の進行状況 | ✓ |
| 備考 | メモ・補足情報 | - |

**ステータスが「落札」または「当社受注見込み」の場合、追加で以下を入力します。**

| 項目 | 説明 |
|------|------|
| 落札金額 / 受注金額概算 | 案件一覧に登録する際の契約金額として使用します |
| 単価契約（0円でも登録可） | 単価契約の場合はチェックを入れると、0円でも登録できます |

---

### 2. ステータスの種類

| ステータス | 説明 |
|------------|------|
| 入札予定 | これから入札が行われる案件 |
| 落札 | 入札で落札した案件 |
| 不落札 | 落札できなかった案件 |
| 当社受注見込み | まだ正式決定前だが、当社受注の見込みが高い案件 |

---

### 3. 一覧表示・フィルタ・ソート

- **一覧表示**：登録した入札スケジュールを一覧で確認できます
- **検索**：案件名・発注者で絞り込み可能
- **フィルタ**：ステータスで絞り込み可能（すべて / 入札予定 / 落札 / 不落札 / 当社受注見込み）
- **ソート**：入札日の昇順・降順で並べ替え可能

---

### 4. 編集・削除

- **編集**：一覧のカードをクリックすると編集モーダルが開き、内容を変更できます
- **削除**：削除ボタンで削除できます。削除時に確認ダイアログが表示されます

---

### 5. 案件一覧への追加

「落札」または「当社受注見込み」の案件は、案件一覧（案件管理）に登録できます。

- **「📋 案件一覧に追加」ボタン**：一覧画面または編集モーダルから実行可能
- **「📋 受注 → 案件一覧に追加」ボタン**：編集モーダル内で、保存と同時に案件一覧へ追加も可能

案件一覧に追加した後は、その案件が案件管理で原価管理・進捗管理の対象となります。

---

## データの保存

入札スケジュールのデータは Supabase に保存され、ログインしたユーザーごとに管理されます。

---

## 補足

- **単価契約**：落札金額や受注金額概算が0円になる単価契約の場合は、「単価契約（0円でも登録可）」にチェックを入れることで登録できます
- **予定金額について**：セキュリティの観点から、入札前の予定金額（予定価格）の入力・表示は行っていません。金額の入力は落札／当社受注見込み時のみです
